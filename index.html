Features:

Unity 2D FTG game. Self-made textures. Two playable characters: The Gentleman and The Evil Snowman. Support two-player play. 3 different game modes.

Instructions:

To play the demo: Download the .rar and unzip. Press Enter to Start. 1P: W-A-S-D to Move. Right_Shift to Attack/Confirm. Right_Control to Dodge(When using the Gentleman) 2P: Arrow Keys to Move. Left_Shift to Attack/Confirm. Left_Control to Dodge(When using the Gentleman)

To view the source code: Download the project folder and open with Unity.(Require Unity 5.2.2 or higher)
